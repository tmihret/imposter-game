# 🕵️ Impostor

A social deduction party game built with SwiftUI and Firebase.

Players are given a secret word — but one player secretly receives a different word without knowing they're the impostor. Through discussion and voting, the group tries to figure out who doesn't belong.

## Modes
- **Local / Pass & Play** — one device, pass the phone around
- **Online Multiplayer** — join with a 6-digit code, chat and vote in real time

## Built With
- SwiftUI
- Firebase Firestore
- Firebase Authentication
